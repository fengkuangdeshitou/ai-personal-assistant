## 💡 额外调节选项

### VS Code 内置快捷键
- `Cmd + +` - 临时放大 (重启后恢复)
- `Cmd + -` - 临时缩小 (重启后恢复)
- `Cmd + 0` - 重置到配置值

### 如果还需要更大
```bash

# 在 VS Code 设置中搜索 "zoom" 并调整数值
# 例如: "window.zoomLevel": 0.5 (放大50%)
```
