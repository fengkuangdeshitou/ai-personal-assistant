## ⚠️ 注意事项

1. **系统要求**: Windows 7/8/10/11 (64位)
2. **首次运行**: 可能会有 Windows 安全警告，点击"允许"即可
3. **网络访问**: 应用需要访问后端API，确保网络连接正常
4. **文件大小**: 绿色版约84MB (最新优化)
5. **路由修复**: 已修复绿色便携版路由显示问题，现在使用HashRouter确保在文件协议下正常工作

## 🔧 故障排除

### 无法启动
- 检查 Windows 版本兼容性
- 尝试以管理员身份运行
- 查看 Windows 事件查看器中的错误信息

### 网络连接问题
- 确保可以访问 `http://localhost:5178` (后端API)
- 检查防火墙设置

### 卸载问题
- 使用 Windows 的"添加/删除程序"卸载
- 或运行安装目录下的 `uninstaller.exe`
