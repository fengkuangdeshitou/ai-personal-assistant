# 🚀 AI 私人助理使用指南

## 🎯 快速开始

### 1. 启动 AI 助理主控制台
```bash
ai
```
这将打开一个交互式菜单，提供以下功能：
- 新建项目
- 项目管理  
- 代码分析
- 自动备份
- 环境检查
- 快速部署
- 学习记录
- 设置助理
- 帮助文档

### 2. 检查开发环境
```bash
ai-env
```
全面检查您的开发环境状态，包括：
- 系统信息
- 开发工具 (Node.js, Python, Java, Git 等)
- 项目目录状态
- 网络连接
- 磁盘空间
- 权限配置

### 3. 查看项目状态报告
```bash
ai-status
```
详细分析所有项目：
- 项目统计 (总数、类型分布)
- Git 状态检查
- 未提交更改警告
- 依赖健康度
- 磁盘使用分析
- 最近活跃项目

### 4. 智能项目备份
```bash
ai-backup
```
智能备份您的项目：
- 全部备份 / 选择性备份 / 重要项目备份
- 自动排除不必要文件 (node_modules, build, .git 等)
- 可选压缩备份
- 自动清理旧备份

### 5. 智能提醒和建议
```bash
ai-remind
```
获取个性化建议：
- 当前时间的智能提醒
- 今日任务概览
- 基于项目状态的建议
- 工作效率优化建议
